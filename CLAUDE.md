# Scrum Toolkit — Chrome Extension

Chrome extension providing Scrum/agile project management tools. Manifest V3.

## Structure
```
Scrum-toolkit/
├── manifest.json
├── Index.html
├── background/
├── content/
├── popup/
├── onboarding/
├── docs/
├── assets/
├── defaultKeywords.json
├── complete-scrum-toolkit.html
├── project-report.html
├── privacy.html
└── terms.html
```

## Loading in Chrome
1. `chrome://extensions` → Developer mode on → Load unpacked
2. Select this folder
